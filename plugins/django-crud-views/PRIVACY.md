# Privacy Policy — django-crud-views (Claude Code plugin)

_Last updated: 2026-07-16_

The `django-crud-views` plugin is **documentation only**. It bundles a single Claude Code skill
(`SKILL.md` and Markdown reference files) that provides guidance for building Django CRUD
interfaces with the `django-crud-views` package.

## Data collection

This plugin **collects, stores, and transmits no user data.**

- It contains **no** MCP servers, hooks, commands, agents, executable code, or network calls.
- It has no telemetry, analytics, tracking, or logging.
- It does not read, upload, or share your code, files, prompts, or any other information.

The plugin is inert content that Claude Code reads locally. Any processing of your prompts and
code is performed by Claude Code and Anthropic's models under
[Anthropic's privacy policy](https://www.anthropic.com/legal/privacy) — not by this plugin.
